# Swing Trade 选股

持仓周期 2-20 天，技术面驱动。基于动量、量价、形态和新闻情绪，从候选池中筛选高概率 swing 买点。

## 输入数据

系统会自动投喂一个 JSON，你**无需查询数据库**。结构如下：

```
{
  "date": "交易日",
  "source": "built-in criteria (...)" 或 "screener (N tickers)",
  "candidates_count": N,
  "candidates": [
    {
      "ticker", "company_name", "sector",
      "close", "ma_20", "ma_50", "ma_200",
      "rsi_14", "vol_ratio", "atr_pct", "pct_chg",
      "dist_ma20_pct", "dist_ma50_pct", "high_20",
      "next_earnings_date", "pe_ratio", "revenue_growth_yoy",
      "recent_news": [ {"headline","sentiment_label","sentiment_score","published_at"} ]  // 近7天，≤3条
    }
  ]
}
```

字段口径：
- `dist_ma20_pct` = (close-MA20)/MA20，正=在均线上方
- `vol_ratio` = 当日量 / 20日均量，>1.5 明显放量
- `atr_pct` = ATR14/close，日均波动，用于止损/目标
- `high_20` = 20日高点，close ≥ high_20×0.99 视为突破
- 基本面字段（pe/营收增速）和情绪可能为 null，缺失按中性处理

**数据来源差异（重要）：**
- `source = built-in`：候选池已预筛（close>MA20 且 >MA50、MA20>MA50、vol_ratio>1.2、RSI 45-75、贴近 MA20）。你**不必再做趋势初筛**，直接进入形态分类。
- `source = screener`：候选**未经趋势过滤**。先剔除不满足「close>MA50 且 ma_50>ma_200」的标的，再分类。

---

## 分析流程

### Step 1：形态分类

把每只候选股归入以下形态之一（互斥，按优先级取第一个匹配）：

**形态 A — 放量突破新高**
- close ≥ high_20 × 0.99
- vol_ratio ≥ 1.5

**形态 B — 强势浅回调（贴 MA20 企稳）**
- dist_ma20_pct ∈ [-0.02, 0.025]（负值仅在 screener 源出现；built-in 源为 0~0.025 的浅回调）
- dist_ma50_pct > 0.03（中期趋势仍在）
- rsi_14 ∈ [45, 60]

**形态 C — 趋势内整理**
- 不满足 A/B，但 close > ma_20 > ma_50 且 rsi 50-70
- 视为「跟踪观察」，评分上限封顶 B

不归任何形态 → 丢弃。

---

### Step 2：新闻情绪

用 `recent_news`（近 7 天）：
- 均分 < -0.3，或存在单条 score < -0.3 的强负面 → 排除
- 均分 > 0.2 → 加分
- 无新闻（中小盘常见）→ 以技术面为主，不扣分

---

### Step 3：财报与估值风险

- `next_earnings_date` 在 7 天内 → 标记「财报风险」，评分上限封顶 B（避免裸扛财报）
- `pe_ratio > 50 且 revenue_growth_yoy < 0.1` → 标记「高估低成长」，扣分
- `revenue_growth_yoy > 0.2` → 加分

---

### Step 4：量化评分（0-100）

| 维度 | 满分 | 评分标准 |
|------|------|----------|
| 形态强度 | 30 | A=30；B=24；C=15 |
| 量能 | 25 | vol_ratio≥2 →25；1.5-2 →20；1.2-1.5 →12；<1.2 →5 |
| 动量(RSI) | 20 | RSI 55-68 →20；50-55 或 68-72 →14；45-50 →8；>75 超买 →0 |
| 情绪 | 15 | 均分>0.2 →15；-0.2~0.2 或无数据 →8；<-0.2 →0 |
| 风险扣减 | 10 | 无财报/估值风险 →10；财报 7 天内 →3；高估低成长 →0 |

评级：**A ≥ 78**（强烈关注）/ **B 60-77**（值得跟踪）/ **C < 60**（观察）。
财报风险或超买（RSI>75）一律封顶 B，并标注「等回调」。

## 注意事项

- 每日收盘后运行，筛出次日候选
- 技术形态是概率游戏，不是确定性
- 每笔交易前检查财报日期，避免裸扛财报
- 止损是硬规则，不因「感觉会反弹」而取消
- 止损/目标用 atr_pct 折算成具体价位，别只给百分比
- 中小盘新闻覆盖低时以技术面为主判断
- 数据日期注意：周末/假期显示上一交易日，属正常
