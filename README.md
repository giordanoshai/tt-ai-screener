# TT AI Screener

A self-hostable MCP server for personal stock analysis. Connect it to Claude.ai and talk to your own stock database. **Claude is the UI.**

Built-in skills:
- **Long-term screening** — find high-growth, high-margin stocks above the 200-day MA
- **Swing trade screening** — find breakout / pullback setups with volume confirmation
- **Position review** — technicals + fundamentals + news + P&L for your holdings
- **Watchlist analysis** — batch analysis of all tracked tickers

Data sources: [yfinance](https://github.com/ranaroussi/yfinance) (free, no key) + [Finnhub](https://finnhub.io) (free key required). Local database: [DuckDB](https://duckdb.org) (no server needed). Optional AI integration for news sentiment and skills analysis (any OpenAI-compatible API).

---

## Requirements

- Python 3.10+
- A free [Finnhub API key](https://finnhub.io/register)
- Claude.ai account (Pro or above for MCP support)

## Quick Start

```bash
git clone https://github.com/giordanoshai/tt-ai-screener.git
cd tt-ai-screener

python -m venv .venv
source .venv/bin/activate        # Mac / Linux
# .venv\Scripts\activate         # Windows
pip install -r requirements.txt

cp .env.example .env             # Edit: add your FINNHUB_KEY
python main.py                   # Start server
```

Open `http://localhost:8766` — add tickers via the web UI, then connect Claude.ai.

## Connect to Claude.ai

### Remote (Claude.ai web — recommended)

```bash
python main.py
# Prints:
#   Web dashboard:  http://localhost:8766
#   MCP endpoint:   http://localhost:8766/mcp?token=<auto-generated>
```

In Claude.ai → Settings → MCP → Add custom integration, paste the MCP endpoint URL.

### Local (Claude Desktop)

```bash
python main.py --mcp
```

Add to `claude_desktop_config.json`:

```json
{
  "mcpServers": {
    "trading": {
      "command": "/path/to/.venv/bin/python",
      "args": ["/path/to/tt-ai-screener/server.py"]
    }
  }
}
```

## Usage

```bash
python main.py                   # Web + MCP remote (default, port 8766)
python main.py --mcp             # MCP stdio only (for Claude Desktop)
python main.py --update          # Run data update once and exit
python main.py --cron 17:30      # Web + MCP + daily auto-update at 5:30 PM
python main.py --port 9000       # Custom port
```

## Web Dashboard

The built-in web UI at `http://localhost:8766` provides:

- **Home** — system status, AI model overview
- **Watchlist** — add/remove tickers, view price + RSI + daily change
- **Screener** — multi-dimensional stock screening with sector, technical, and fundamental filters
- **Settings** — AI model configuration, data update scheduling, language preference

---

## AI Configuration (Optional)

AI powers two features: news sentiment scoring (during data updates) and skills analysis (watchlist/position analysis).

Configure via **Web Settings → AI Models**, or seed from `.env` on first run:

```bash
# .env (optional — only used for initial database seed)
SENTIMENT_API_BASE=https://api.openai.com/v1
SENTIMENT_API_KEY=your_key
SENTIMENT_MODEL=gpt-4o-mini

AI_API_BASE=https://api.openai.com/v1
AI_API_KEY=your_key
AI_MODEL=gpt-4o
```

Any OpenAI-compatible API works (OpenAI, Anthropic, Alibaba DashScope, etc.).

## MCP Tools

| Tool | Description |
|---|---|
| `screen_stocks` | Parameterized stock screener (longterm/swing, sector, PE, RSI, MA filters) |
| `list_watchlist` | List watchlist with price, change, technicals |
| `add_to_watchlist` | Add ticker to watchlist |
| `remove_from_watchlist` | Remove ticker from watchlist |
| `analyze_watchlist` | AI analysis of all watchlist tickers |
| `list_positions` | List positions with P&L |
| `add_position` | Add/update a position (ticker, cost, shares) |
| `remove_position` | Remove a position |
| `analyze_positions` | AI analysis of all positions |

## Data Sources

| Data | Source | Cost |
|---|---|---|
| OHLCV (2 years daily) | yfinance | Free |
| Technical indicators (RSI, MA, ATR) | Computed locally | Free |
| Fundamentals (PE, margins, growth, analyst ratings, earnings, insider MSPR) | Finnhub | Free tier |
| News (30 days) | Finnhub | Free tier |
| News sentiment | AI API (optional) | Pay per use |

Finnhub free tier: 60 API calls/minute. Premium: 300+/minute (set `FINNHUB_TIER=premium` in `.env`).

---

## Privacy

All data is stored locally in `trading.duckdb`. External requests go only to yfinance, Finnhub, and your configured AI API (if any). Your portfolio and trade records never leave your machine.
